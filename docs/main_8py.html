Detailed Description

This file contains the main file, modified from basic_tasks.py, to run two tasks sequentially in a real-time scheduler.

Each task is the main file from the previous Lab, Lab03 - control loop. The tasks are modified as generator files, and the main file in this task runs each task after one another.
